CSIAOnline Application - Edge Shortcut
** Installation Manual **
Desktop
1. 우상단 ...을 누른다.

2. `앱`을 누른다.

3. `이 사이트를 앱으로 설치`를 누른다.

4. 이름을 지정하고 `설치`를 누른다.

5. 추가 설정을 할 수 있다.
`작업 표시줄에 고정` 또는 `바탕 화면 바로 가기 만들기`를 추천한다.

6. 앱을 열면 나오는 창에 우상단 ...을 누르면 추가 정보가 뜬다.
앱 설정을 누르면 앱 설정에 들어갈 수 있다.

7. 다양한 앱 설정을 만질 수 있다.

8. 바탕 화면에 생성된 모습이다.

Mobile
1. 하단 설정을 누른다.

2. 옆으로 드래그한다.

3. `휴대폰에 추가`를 누른다.

4. 이름을 지정하고 `추가`를 누른다.

5. `추가`를 누른다.

6. 홈 화면에 만들어진 모습이다.
