CSIAOnline Application - Chrome Shortcut
** Installation Manual **
Desktop
1. 우상단 설정을 누른다.

2. `도구 더보기`을 누른다.

3. `바로가기 만들기`를 누른다.

4. 이름을 지정하고 `만들기`을 누른다.

5. 성공적으로 만들어진 화면이다.

6. 바탕화면에 추가된 화면이다.

Mobile
1. 최상단 설정을 누른다.

2. `홈 화면에 추가`를 누른다.

3. 원하는 이름을 지정하고 `추가`를 누른다.

4. 계속 `추가`를 누른다.

5. 홈 화면에 설치된 모습이다.
