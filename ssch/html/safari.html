CSIAOnline Application - Safari Shortcut
** Installation Manual **
Desktop
1. 사이트에 접속한다.

2. 상단 링크를 클릭한다.

3. Command + Option 키를 누른 상태로 링크를 클릭한 후 바탕화면에 드래그한다.

4. 설정을 통해 원하는 로고나 이름을 지정할 수 있다.

Desktop
내용 보기 (click)
Mobile
1. Safari 최하단 중간의 공유 버튼을 누른다.

2. 밑으로 스크롤한다.

3. `홈 화면에 추가`를 누른다.

4. 원하는 제목을 설정하고 추가를 누른다.
